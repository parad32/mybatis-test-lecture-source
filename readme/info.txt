콘솔을 활용해 카페 키오스크 프로그램을 제작하시오.
구현된 프로그램은 아래 '개발 요구사항'과 '기능 요구사항'에 기술되어 있는 내용을 모두 만족해야 한다.

<개발 요구사항>
1. Java 애플리케이션 개발 시, MyBatis 프레임워크를 사용할 것.
2. MVC-II 패턴을 적용할 것.
3. MyBatis 프레임워크 설정은 아래와 같이 진행:
  1) MyBatis Configuration 설정은 Java 언어로 구현
  2) MyBatis Mapper 설정은 외부 XML 파일로 구현(MenuMapper.xml)
4. 데이터베이스 스키마는 수업 시간에 사용한 menudb를 그대로 이용할 것.
5. 결과물 제출
  1) 구현한 콘솔 프로젝트를 개인 GitHub public Repository에 업로드 한 뒤 주소를 답안에 제출하시오.
  2) 구현한 콘솔 프로젝트를 압축 파일로 압축('20240802_송언석.zip')하여 업로드하시오.

<기능 요구사항>
1. 전체 메뉴 조회
2. 특정 메뉴 조회
3. 신규 메뉴 추가
4. 기존 메뉴 정보 수정
5. 기존 메뉴 삭제
